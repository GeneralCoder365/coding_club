# INSTRUCTIONS FOR ZOOM_FAKE_CAMERA #

## SETUP ##

### Zoom Settings ###
#### Make sure that in your Zoom audio settings, the following options are turned ON: ####
* Audio:
- [x] Automatically join audio by computer when joining a meeting
- [x] Mute my microphone when joining a meeting

* General:
- [x] Enter full screen automatically when starting or joining a meeting

#### Make sure that in your Zoom audio settings, the following options are turned OFF: ####
* Video:
- [ ] Turn off my video when joining meeting
- [ ] Always show video preview dialog when joining a video meeting

#### Make sure that the dialog box that rechecks if you want to join a Zoom meeting is unchecked so that it joins by default ####

#### OBS Camera: Open OBS and create new VLC Video Source and save it in C\Users\{UserName}\Videos | Make sure the loop video option is checked in the video properties  |  Download VirtualCam from https://obsproject.com/forum/resources/obs-virtualcam.949/ and set it up | Go back to OBS and under Tools, click VirtualCam, click Start and check the box for AutoStart (make sure the Target Camera is OBS-Camera) ####
